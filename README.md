<h1 align="center">📱 Lista de Contatos</h1>

<p align="center">
  Aplicação de terminal em Python para gerenciamento de contatos com operações de CRUD.
</p>

<hr>

<h2>🚀 Sobre o projeto</h2>

<p>
Este projeto foi desenvolvido como desafio do <strong>primeiro módulo da Rocketseat</strong>, sendo construído do absoluto zero com o objetivo de praticar conceitos fundamentais da linguagem Python.
</p>

<p>Durante o desenvolvimento, foram aplicados na prática:</p>

<ul>
  <li>📌 Estrutura de CRUD (Criar, listar, editar e excluir contatos)</li>
  <li>📌 Construção de menus interativos no terminal</li>
  <li>📌 Validação e tratamento de dados de entrada</li>
  <li>📌 Tratamento de erros para evitar falhas no programa</li>
  <li>📌 Organização de código em módulos</li>
</ul>

<hr>

<h2>⚙️ Funcionalidades</h2>

<ul>
  <li>➕ Criar novos contatos</li>
  <li>📋 Listar todos os contatos</li>
  <li>⭐ Listar contatos favoritos</li>
  <li>✏️ Editar informações de um contato</li>
  <li>🗑️ Excluir contatos</li>
  <li>📞 Validação e formatação de número de telefone</li>
  <li>📧 Validação de email</li>
</ul>

<hr>

<h2>🧠 Aprendizados</h2>

<ul>
  <li>Estruturas de repetição (<code>while</code>)</li>
  <li>Condicionais (<code>if</code>, <code>elif</code>, <code>else</code>)</li>
  <li>Funções e modularização</li>
  <li>Boas práticas de organização de código</li>
  <li>Interação com o usuário via terminal</li>
</ul>
